✨ Key Features

Add Resources

Add Books, Magazines, and DVDs with unique attributes like ISBN, issue numbers, and duration.

Register Members

Register new library members with auto-generated IDs and track their borrowing history.

Borrow & Return

Seamless borrowing and returning system with real-time availability tracking.

Filter & Search

Filter resources by type (Books, Magazines, DVDs) and search functionality.

Late Fee Calculation

Automatic late fee calculation based on resource type (Books: Rs. 5/day, Magazines: Rs. 3/day, DVDs: Rs. 10/day).

Data Persistence

All data is saved to file and automatically loaded on application startup.

🎓 OOP Concepts Demonstrated

Inheritance

LibraryResource (base class) → Book, Magazine, DVD (derived classes)

Polymorphism

Virtual functions: getType(), getExtraInfo(), calculateLateFee(), toJSON()

Encapsulation

Private data members with public getter/setter methods in LibraryMember and Library classes

Abstraction

Base class LibraryResource provides abstract interface; Library class manages complex operations
